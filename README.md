## Description

参考搭建nest-cli搭建框架
[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.


模仿多点商城，电商系统

技术栈：
[x] NestJS
[x] TypeScript
[x] MySQL
[x] docker

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run dev
或
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
